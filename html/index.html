Estrutura básica

Toda tag no HTML tem um <> e uma tag dentro, tudo que estiver dentro será HTML.

Tags

title: nome da guia

body: textos que aparecem pro usúario

i: itálico

strong: negrito

input: campo para entrada de dados(textos,números,etc)

img: tag para colocar imagens no site

u: sublinhado

mark: grifar um texto

sup: deixar um texto na parte superior

hr: cria uma linha embaixo

Atributo das Tags

id: adiciona um identificador

input type: diz qual o tipo para entrada de dados.

Exemplo input type="text", só será recebida entrada de dados do tipo texto

img: src: endereço da imagem que vai ser inserida. width: tamanho da imagem

a href="" target="" title="". href: referencia para qual endereço vai levar

target: se tiver com _self ele vai abrir o link na própria pagina, se tiver _blank abre em uma nova guia

title: texto em cima do link (acessibilidade)

Textos

H's e seus niveis, h1 ate o h6 para títulos e subtítulos

p para paragráfos

blockquote: para citaçôes dando uma margem na página

small: cria um texto pequeno

Listas

ol: usado para criar listas organizadas em ordem númerica. necessita do li para funcionar

ul para marcadores ao invés de números (bolinhas)

Links

a: ancora que te leva para outros endereços

Como navegar no próprio site

Criar um novo arquivo e adicionar na âncora

a href="arquivo" target="_self" title=""

Ancoragem (Clicar em um link e continuar na mesma página)

na tag a colocar uma # com o nome do id

a href="#inicio"
